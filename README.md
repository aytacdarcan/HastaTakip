# PediaTrack

A full-stack pediatric patient tracking system developed with ASP.NET Core Web API and Blazor Server.

The system is designed to manage patient records, visits, anthropometric measurements, growth analysis and laboratory results from a centralized clinical interface.

## Features

- Patient registration and visit management
- Anthropometric measurement tracking
- Growth chart visualization
- Laboratory result management
- REST API architecture
- Modular clinical management interface

## Growth Analysis

PediaTrack uses Neyzi growth reference datasets and LMS-based percentile calculations for pediatric growth analysis and visualization.

## Technologies Used

- ASP.NET Core Web API
- Blazor Server
- C#
- SQL Server
- Entity Framework Core
- Chart.js
- HTML / CSS / JavaScript

## Project Purpose

This project was developed to practice full-stack web application development, clinical data management, API design and interactive data visualization.

## Status

This project is actively being developed and improved.
